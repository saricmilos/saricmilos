How to Use the VSC Modeler

1

Drag Components

Drag electrical components from the left panel onto the canvas to create your circuit.

2

Connect Elements

Use the connection tool to connect components and create your complete circuit.

3

Configure Properties

Select any component to edit its parameters and properties in the right panel.

4

Run Simulation

Click "Simulate" to run your model and view voltage, current, and power results.
